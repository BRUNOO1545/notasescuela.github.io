Sobre

Notas Escuela es un software el cual le permitira administrar notas de alumnos y promediar las mismas

Finalidad

Este se especializa en sacar notas anuales del todal de las materias con el fin de facilitar el uso del papel, solo teniendo el DNI del alumno este podra acceder a todos los datos previamente ingresados y modificarlos segun la situacion lo requiera

Como Funciona

Gracias a la tecnologia de Firebase que proporciona Google, Notas Escuela hace uso de Realtime Database ya que permite que el usuario con muy poca conexion a internet pueda resolver los datos solicitados y en caso no contar con la misma, este se quedara a la espera de internet
